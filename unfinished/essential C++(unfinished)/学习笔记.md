## C++引用
+ 把引用作为参数,，起到和指针一样的作用，看起来比指针清楚
+ 把引用作为返回值，让函数返回值能被想变量一样被赋值
```
void swap(int& x, int& y){
    int temp;
    temp = x;
    x = y;
    y = temp;

    return;
}
```
